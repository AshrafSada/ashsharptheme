## Installation

1. Open **Extensions** sidebar panel in VS Code. `View â†’ Extensions`
2. Search for `Ash Sharp Theme`
3. Click **Install**
4. Click **Reload**
5. File > Preferences > Color Theme > **Ash Sharp Theme**
